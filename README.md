# Quiz 
Projeto fullstack com Next.js de uma aplicaçção de perguntas e respostas, de conhecimentos gerais.

## Como rodar o projeto

```bash
npm run dev
# ou
yarn dev
```

## Próximos passos

- definição de quantidade de perguntas pelo usuário
- página inicial
- pular para a próxima pergunta ao responder
- correções nas perguntas
- melhorias na estilização
